# Amazon Secrets Manager Secret Version Data Source

The Amazon Secrets Manager Secret Version provides information about a Secrets Manager secret version,
including its secret value.

-> **Note:** Data sources is a feature exclusively to HCL2 templates.

Basic examples of usage:

```hcl
data "amazon-secretsmanager-secret-version" "basic-example" {
  id = "packer_test_secret"
  version_stage = "example"
}
```
