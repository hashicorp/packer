# Amazon Secrets Manager Secret Data Source

The Amazon Secrets Manager Secret provides metadata information about a Secrets Manager secret

-> **Note:** Data sources is a feature exclusively to HCL2 templates.

Basic examples of usage:

```hcl
data "amazon-secretsmanager-secret" "basic-example" {
  arn = "arn:aws:secretsmanager:us-east-1:123456789012:secret:example-123456"
}
```

```hcl
data "amazon-secretsmanager-secret" "basic-example" {
  name = "packer_test_secret"
}
```
